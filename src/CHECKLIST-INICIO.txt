  PARTE 1: CONFIGURACIÓN INICIAL (SOLO UNA VEZ)
═════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 1. Verificar Node.js instalado                                 │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal:
   → node --version
   → npm --version
   
   ¿Aparecen números de versión?
   ✅ Sí → Continúa al paso 2
   ❌ No → Instala Node.js: https://nodejs.org/

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 2. Abrir proyecto en Visual Studio Code                        │
└─────────────────────────────────────────────────────────────────────┘

   Desde la terminal:
   → code .
   
   O manualmente:
   → Abre VS Code
   → File → Open Folder
   → Selecciona la carpeta del proyecto

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 3. Instalar extensiones recomendadas                           │
└─────────────────────────────────────────────────────────────────────┘

   VS Code mostrará una notificación:
   "Do you want to install the recommended extensions?"
   
   → Click en "Install All" o "Install"
   
   Si no aparece:
   1. Ctrl + Shift + X (abrir extensiones)
   2. Buscar e instalar:
      • ESLint
      • Prettier
      • Tailwind CSS IntelliSense
      • ES7+ React Snippets

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 4. Instalar dependencias del Frontend                          │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal de VS Code (Ctrl + Ñ):
   → npm install
   
   Espera 1-2 minutos...
   
   ✅ Debe decir: "added XXX packages"

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 5. Verificar que no hay errores                                │
└─────────────────────────────────────────────────────────────────────┘

   1. Abre cualquier archivo .tsx
   2. ¿Aparecen líneas rojas de error?
      
      ❌ Sí → Ejecuta:
         Ctrl + Shift + P
         Escribe: "TypeScript: Restart TS Server"
         Enter
      
      ✅ No → ¡Perfecto! Continúa

═════════════════════════════════════════════════════════════════════════
  PARTE 2: INICIAR FRONTEND (CADA VEZ QUE TRABAJES)
═════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 6. Iniciar el servidor de desarrollo                           │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal (Ctrl + Ñ):
   → npm run dev
   
   Deberías ver:
   
   ╭──────────────────────────────────────────────╮
   │                                              │
   │   VITE v6.x.x  ready in XXX ms              │
   │                                              │
   │   ➜  Local:   http://localhost:5173/        │
   │   ➜  press h + enter to show help           │
   │                                              │
   ╰──────────────────────────────────────────────╯

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 7. Verificar que la aplicación carga                           │
└─────────────────────────────────────────────────────────────────────┘

   El navegador debería abrir automáticamente.
   Si no, abre manualmente: http://localhost:5173
   
   ¿Se ve la pantalla de login de Huella?
   ✅ Sí → ¡Listo para trabajar!
   ❌ No → Revisa la terminal por errores

═════════════════════════════════════════════════════════════════════════
  PARTE 3: USO DIARIO
═════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 8. Registrar primer usuario                                    │
└─────────────────────────────────────────────────────────────────────┘

   Primera vez usando la app:
   
   1. Click en "Registrarse"
   2. Completa el formulario:
      • Nombre completo
      • Email
      • Contraseña (mínimo 6 caracteres)
      • Confirmar contraseña
      • Rol: Administrador
   3. Click "Crear Cuenta"
   4. Inicia sesión

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 9. Familiarizarse con la interfaz                              │
└─────────────────────────────────────────────────────────────────────┘

   Como Administrador puedes:
   • Ver Dashboard con estadísticas
   • Gestionar Personas
   • Gestionar Usuarios
   • Ver Reportes
   • Configurar sistema
   • Chatear con Guardas

  PARTE 4: (OPCIONAL) CONFIGURAR BACKEND CON MYSQL
═════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 10. Instalar MySQL                                             │
└─────────────────────────────────────────────────────────────────────┘

   📖 Guía completa en: MYSQL-WORKBENCH-PASO-A-PASO.txt
   
   Resumen:
   1. Descargar MySQL: https://dev.mysql.com/downloads/installer/
   2. Instalar MySQL Server + Workbench
   3. Configurar password de root
   4. Verificar que esté corriendo

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 11. Crear base de datos                                        │
└─────────────────────────────────────────────────────────────────────┘

   1. Abrir MySQL Workbench
   2. Conectar a localhost
   3. Abrir: backend/database/schema.sql
   4. Ejecutar todo el script (⚡ botón)
   5. Verificar que se creó "huella_sena"

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 12. Configurar variables de entorno                            │
└─────────────────────────────────────────────────────────────────────┘

   1. cd backend
   2. copy .env.example .env (Windows)
      cp .env.example .env (Mac/Linux)
   3. Editar .env con tus credenciales MySQL
   4. Guardar

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 13. Instalar dependencias del Backend                          │
└─────────────────────────────────────────────────────────────────────┘

   cd backend
   npm install
   
   Espera a que termine...

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 14. Iniciar el Backend                                         │
└─────────────────────────────────────────────────────────────────────┘

   En nueva terminal (Ctrl + Shift + Ñ):
   cd backend
   npm run dev
   
   Deberías ver:
   
   ╔═══════════════════════════════════════════════════════════════╗
   ║       🟢 SERVIDOR HUELLA SENA - INICIADO CORRECTAMENTE       ║
   ╚═══════════════════════════════════════════════════════════════╝
   🚀 Servidor corriendo en: http://localhost:5000

┌─────────────────────────────────────────────────────────────────────┐
│  [ ] 15. Conectar Frontend con Backend                              │
└─────────────────────────────────────────────────────────────────────┘

   📖 Guía completa en: CONFIGURACION-BACKEND.md
   
   (Requiere crear servicio API en el frontend)

  🚀 COMANDOS RÁPIDOS DE REFERENCIA
═════════════════════════════════════════════════════════════════════════

Frontend:
  npm install              ← Instalar dependencias
  npm run dev              ← Iniciar desarrollo
  npm run build            ← Compilar para producción

Backend:
  cd backend               ← Ir a carpeta backend
  npm install              ← Instalar dependencias
  npm run dev              ← Iniciar con auto-reload

VS Code:
  Ctrl + Ñ                 ← Abrir terminal
  Ctrl + Shift + Ñ         ← Nueva terminal
  Ctrl + P                 ← Buscar archivo
  Ctrl + Shift + P         ← Paleta de comandos
  Ctrl + B                 ← Mostrar/Ocultar sidebar
  Ctrl + /                 ← Comentar línea

TypeScript:
  Ctrl + Shift + P         ← Abrir comandos
  "TypeScript: Restart TS Server"  ← Reiniciar TypeScript
