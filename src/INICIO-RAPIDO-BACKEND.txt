┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 1: INSTALAR MYSQL (10 minutos)                             │
└─────────────────────────────────────────────────────────────────────┘

   1. Descarga MySQL Installer:
      🌐 https://dev.mysql.com/downloads/installer/
   
   2. Ejecuta el instalador
      → Elige: "MySQL Server" y "MySQL Workbench"
      → Configura password para root (¡GUÁRDALA!)
      → Puerto: 3306 (dejar por defecto)
   
   3. Completa la instalación

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 2: CREAR BASE DE DATOS (3 minutos)                         │
└─────────────────────────────────────────────────────────────────────┘

   1. Abre MySQL Workbench
   
   2. Crea nueva conexión:
      → Host: localhost
      → Port: 3306
      → Username: root
      → Password: (la que configuraste)
   
   3. Conecta y abre: backend/database/schema.sql
   
   4. Click en ⚡ "Execute" (o Ctrl+Shift+Enter)
   
   5. Verifica que se creó la base de datos "huella_sena"

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 3: CONFIGURAR BACKEND (2 minutos)                          │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal:
   
   → cd backend
   
   → copy .env.example .env    (Windows)
     cp .env.example .env      (Mac/Linux)
   
   → Abre el archivo .env y cambia:
   
     DB_PASSWORD=tu_password_mysql
     JWT_SECRET=cambiar_por_algo_aleatorio_123456
   
   → npm install
   
   ⏳ Espera a que termine (1-2 minutos)

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 4: INICIAR BACKEND (10 segundos)                           │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal (carpeta backend):
   
   → npm run dev
   
   Deberías ver:
   
   ╔═══════════════════════════════════════════════════════════════╗
   ║       🟢 SERVIDOR HUELLA SENA - INICIADO CORRECTAMENTE       ║
   ╚═══════════════════════════════════════════════════════════════╝
   🚀 Servidor corriendo en: http://localhost:5000
   
   ✅ ¡Backend funcionando!
   
   ⚠️  IMPORTANTE: Deja esta terminal ABIERTA

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 5: PROBAR QUE FUNCIONA (1 minuto)                          │
└─────────────────────────────────────────────────────────────────────┘

   Opción A - Navegador:
   
   → Abre: http://localhost:5000
   
   Deberías ver mensaje de bienvenida de la API
   
   
   Opción B - Comprobar Login:
   
   → Abre otra terminal (sin cerrar la del backend)
   
   → Ejecuta:
   
   curl -X POST http://localhost:5000/api/auth/login ^
     -H "Content-Type: application/json" ^
     -d "{\"email\":\"admin@huella.com\",\"password\":\"admin123\"}"
   
   Si ves un JSON con "success": true → ¡Funciona! ✅
