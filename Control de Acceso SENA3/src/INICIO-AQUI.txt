📍 ESTÁS AQUÍ: Has descargado el proyecto y necesitas abrirlo en VS Code

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 1: REORGANIZAR ARCHIVOS (5 minutos)                        │
└─────────────────────────────────────────────────────────────────────┘

   📖 Lee el archivo: "reorganizar.md"
   
   Resumen:
   1. Crea la carpeta "src/" en la raíz
   2. Mueve estas 5 carpetas DENTRO de src/:
      • components/
      • data/
      • styles/
      • types/
      • utils/
   3. Borra el archivo "App.tsx" de la raíz (está duplicado)

   💡 Alternativa: Usa los comandos del archivo "comandos-terminal.md"

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 2: INSTALAR NODE.JS (si no lo tienes)                      │
└─────────────────────────────────────────────────────────────────────┘

   🌐 Descarga desde: https://nodejs.org/
   
   Elige: Versión LTS (Recomendada)
   Instala siguiendo el asistente
   
   Verifica la instalación abriendo una terminal y ejecutando:
   → node --version

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 3: ABRIR EN VISUAL STUDIO CODE                             │
└─────────────────────────────────────────────────────────────────────┘

   1. Abre Visual Studio Code
   2. Menú: File → Open Folder
   3. Selecciona la carpeta "sena-control-acceso"
   4. Abre la terminal: Ctrl + Ñ (o Cmd + Ñ en Mac)

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 4: INSTALAR DEPENDENCIAS (2-5 minutos)                     │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal de VS Code, escribe:
   
   → npm install
   
   ⏳ Espera a que termine... verás una barra de progreso

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 5: EJECUTAR EL PROYECTO                                    │
└─────────────────────────────────────────────────────────────────────┘

   En la terminal, escribe:
   
   → npm run dev
   
   Se abrirá automáticamente en: http://localhost:3000/

┌─────────────────────────────────────────────────────────────────────┐
│  ✅ PASO 6: USAR LA APLICACIÓN                                      │
└─────────────────────────────────────────────────────────────────────┘

   Primera vez:
   
   1. Click en "Registrarse"
   2. Completa el formulario:
      • Nombre completo
      • Email
      • Contraseña (mínimo 6 caracteres)
      • Confirmar contraseña
      • Rol: Administrador o Guarda
   3. Click en "Crear Cuenta"
   4. Inicia sesión con tus credenciales
   
   ¡Listo! Ya puedes usar el sistema 🎉
